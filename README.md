## Hirify 🚀

## Build smarter CVs. Get hired faster.

Hirify is an AI-powered CV builder designed to help job seekers create professional, ATS-optimized resumes in minutes. With a clean interface and powerful AI assistance, users can craft resumes that stand out and pass Applicant Tracking Systems used by modern recruiters.

Whether you're a student, a job seeker, or a professional looking to upgrade your resume, Hirify makes the process fast, simple, and effective.

## ✨ Features
## 📝 Free CV Builder

Create professional resumes easily using a guided interface. Users can build and export their CVs for free.

## 🤖 AI-Powered Resume Generation

Generate resume content using Google Gemini AI to craft professional summaries, experience descriptions, and optimized content.

## 📊 ATS Score Analysis

Analyze your CV against common Applicant Tracking System (ATS) standards and get a score with improvement suggestions.

## 🎯 ATS-Optimized Keywords

Premium templates include industry-relevant keywords to improve ATS compatibility and recruiter visibility.

## 🎨 Premium Resume Templates

Access beautifully designed templates optimized for both readability and ATS parsing.

## 📄 One-Click PDF Export

Download your CV instantly as a polished PDF file ready to send to employers.

## 🔐 Secure Authentication

Users can securely sign in using Google OAuth powered by Supabase.

## 💎 Premium Subscription

Upgrade to unlock:

Advanced templates

Enhanced ATS insights

Additional AI resume generation features

## 🏗 Tech Stack
## Framework

Next.js (App Router) with TypeScript

## Frontend

React

Tailwind CSS

## Authentication & Database

Supabase

PostgreSQL database

Row Level Security (RLS)

Google OAuth authentication

@supabase/ssr for cookie-based auth

## AI Integration

Google Gemini API (@google/generative-ai)

Resume content generation

ATS scoring and analysis

## PDF Processing

react-to-pdf – client-side PDF export

pdf-parse – server-side text extraction for ATS analysis

## Payments

Stripe (integration planned)

## Other Tools

PostCSS

Supabase SSR utilities

Sign in with Google.
